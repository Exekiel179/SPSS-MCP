## 📝 配置文件位置

```
你的系统
├── C:\Users\你的用户名\
│   ├── .claude\
│   │   ├── settings.json          ← Claude Code 配置（添加 mcpServers）
│   │   └── skills\                ← Claude Code 技能目录
│   │       ├── spss-analyst\      ← 分析技能
│   │       └── spss-mcp-guard\    ← 防护技能
│   │
│   └── AppData\Roaming\Claude\
│       └── claude_desktop_config.json  ← Claude Desktop 配置（如果使用）
│
├── F:\MCP\spss-mcp\               ← SPSS-MCP 项目目录
│   ├── .env                       ← 环境配置（SPSS 路径等）
│   ├── src\spss_mcp\              ← MCP 服务器代码
│   └── skills\                    ← 技能源文件
│
└── C:\Program Files\IBM\SPSS Statistics\31\  ← SPSS 安装目录
    ├── stats.exe                  ← SPSS 主程序
    └── Python3\python.exe         ← SPSS Python 环境
```

## 🔄 数据流向

### 用户发起分析请求

```
1. 用户输入
   ↓
   "请对 data.sav 进行描述性统计"
   ↓
2. Claude Code 理解意图
   ↓
   识别需要调用 SPSS 工具
   ↓
3. 调用 MCP 工具
   ↓
   spss_descriptives(file_path="data.sav", variables=["age", "income"])
   ↓
4. SPSS-MCP 处理
   ↓
   - 读取 data.sav
   - 生成 SPSS 语法: DESCRIPTIVES VARIABLES=age income.
   - 调用 SPSS 引擎执行
   ↓
5. SPSS 执行分析
   ↓
   - 计算统计量
   - 生成 .spv 文件
   - 返回结果数据
   ↓
6. SPSS-MCP 格式化
   ↓
   - 解析 SPSS 输出
   - 转换为 Markdown 表格
   - 保存 .spv 和 .sps 文件
   ↓
7. Claude Code 展示
   ↓
   | 变量   | 均值  | 标准差 | 最小值 | 最大值 |
   |--------|-------|--------|--------|--------|
   | age    | 35.2  | 8.5    | 22     | 65     |
   | income | 45000 | 12000  | 20000  | 80000  |
   ↓
8. 用户查看结果
```

## 🚨 常见连接问题

### 问题 1: MCP 服务器未连接

```
症状:
┌─────────────────────────────────┐
│ Claude Code 右下角显示:          │
│ ⚠️ spss 服务器未连接             │
└─────────────────────────────────┘

检查清单:
□ settings.json 配置是否正确？
□ 是否重启了 Claude Code？
□ spss-mcp 命令是否可用？
  (在命令行运行: spss-mcp status)
```

### 问题 2: SPSS 未检测到

```
症状:
┌─────────────────────────────────┐
│ > spss-mcp status               │
│ SPSS : ✗ 未检测到               │
└─────────────────────────────────┘

解决方案:
1. 创建 .env 文件
2. 添加: SPSS_INSTALL_PATH=C:\Program Files\IBM\SPSS Statistics\31
3. 重启 Claude Code
```

### 问题 3: 分析执行失败

```
症状:
┌─────────────────────────────────┐
│ Claude: 分析执行失败             │
│ 错误: 超时或语法错误             │
└─────────────────────────────────┘

检查清单:
□ 数据文件路径是否正确？
□ 变量名是否存在？
□ 是否需要增加超时时间？
  (在 .env 添加: SPSS_TIMEOUT=300)
```
